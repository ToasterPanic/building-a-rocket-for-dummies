# Building a Rocket for Dummies

You have the knowledge. You have the framework. But do you have the time? Rush last-second to find parts around a labrynthic storage building and assemble a fully-fledged spaceship in TWO MINUTES, lest you risk losing your job (or worse!)

## Notes
- There is a LOT of modifiers.
- If you get tired of the starting dialogue, there's a modifier in the Settings for that.
- It's possible to beat the game without glitches.
